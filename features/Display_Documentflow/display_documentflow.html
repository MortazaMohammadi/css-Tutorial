Block Element

This is a block element. It takes the full width available and starts on a new line.

Inline Element

This is an inline element. It only takes as much width as necessary and does not start on a new line.

Inline-Block Element

This is an inline-block element. It behaves like an inline element but allows setting width and height.

None Display

This element is hidden using display: none; and will not be visible on the page.

Default Block Tags

Common block-level tags include <div>, <h1> to <h6>, <p>, <ul>, <ol>, and <li>.

Default Inline Tags

Common inline tags include <span>, <a>, <strong>, <em>, and <img>.

Default Inline-Block Tags

Common inline-block tags include <button>, <input>, <select>, and <textarea>.
